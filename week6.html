Week 6 – Performance Testing

Objective

The objective of this phase was to evaluate system performance under load and observe CPU, memory, and process behaviour using Linux monitoring tools.

Testing Tools Used

  • s-tui – CPU stress testing and temperature monitoring
  • htop – Real-time process and memory monitoring
  • iostat – Disk I/O performance analysis

Performance Testing Evidence

The screenshot below shows CPU load and temperature monitoring using s-tui while the system was under stress. This confirms that the system was successfully tested under load conditions.

Figure: CPU stress testing using s-tui, showing real-time usage, temperature, and frequency across CPU cores.

Reflection

Performance testing provided valuable insight into how the system behaves under heavy load. Monitoring CPU temperature, utilization, and system responsiveness ensured the system could safely operate within acceptable limits. This step verified that earlier configuration and security decisions did not negatively impact system performance.
