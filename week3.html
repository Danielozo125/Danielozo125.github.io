Week 3 – Application Selection for Performance Testing

Selected Applications

  • Stress-ng – CPU and system stress testing
  • Memtester – RAM testing
  • FIO – Disk I/O benchmarking
  • Iperf3 – Network throughput testing
  • Game Server – Simulated real-world workload

Installation

Expected Resource Usage

Each tool targets a specific system component:

  • CPU stress via stress-ng
  • Memory validation using memtester
  • Disk performance measured with fio
  • Network throughput tested using iperf3

Reflection

This stage ensured that system performance could be measured accurately before moving on to advanced configuration and optimisation tasks.
