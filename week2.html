Week 2 – Security Planning and Testing Methodology

Performance Testing Plan

Remote monitoring will be conducted via SSH using command-line tools such as top, htop, vmstat, and iostat to measure system performance.

Security Configuration Checklist

  • SSH hardening
  • Firewall configuration
  • User privilege management
  • Automatic updates
  • Network security

Threat Model

Identified threats include brute-force SSH attacks, unauthorised access, and service exploitation. Mitigation strategies include key-based SSH, firewalls, and fail2ban.

Reflection

This phase ensured security considerations were planned before deployment.
